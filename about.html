My little Zoom clone
...

This is one of my proudest projects. Having gone through Rocket Academy, I had the confidence in myself to to try develop my small version of Zoom. I spent about 5 days reading documentations and running small tests before finally deciding i could get this done by due date. Although there were plenty of react libraries for video applications, i chose to dig in myself and worked with the Web API's MediaDevices to get the each user's stream. Good choice i must say, as it gave me the opportunity to pick up new technologies while understanding them myself.

Challenges faced

Obviously there were many challenges/blockers i faced while developing this web app, but i think the most interesting one was figuring out how to host a group video call. Handling a 1-to-1 was quite straight forward as all i had to do was get one user to send a request and handle the anwsering logic on the other end. But with many other users doing the same thing i got overwhelmed by the idea ðŸ¤¯ Turns out it was quite straight forward! I made it so that in a group of let's say 3 (person "A", "B" and "C"), the first person (presumably the host whom we'll call "A"), would only have to care about sending out the call request. The other two users, "B" and "C" would each receive a call request from the host (person "A") and the recieving logic would run on their end respectively.
